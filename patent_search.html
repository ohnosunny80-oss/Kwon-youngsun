📂 특허 데이터 탭 (추천)
링크를 열면 자동으로 특허 데이터를 불러옵니다.
업체 버튼 클릭 → 특허 목록 → 청구항 복사 → HWP 붙여넣기
🔄 데이터 업데이트 방법
1. companies.xlsx에 업체 추가
2. CMD: py kipris_final_v10.py 실행
3. patent_results.json → GitHub에 업로드
4. 이 링크 새로고침
⚠ 로컬 서버(py kipris_server.py)가 실행 중일 때 정확하게 작동합니다.
기업명을 입력하고 조회하세요
